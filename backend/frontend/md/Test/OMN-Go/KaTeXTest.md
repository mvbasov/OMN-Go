#### The simplest test

$$a_b$$

#### Real example

$$
  \begin{aligned}
&\text{Given a quadratic equation:} \\[6pt]
&ax^2 + bx + c = 0, \quad a \neq 0 \\[10pt]
&\text{Divide both sides by } a: \\[6pt]
&x^2 + \frac{b}{a}x + \frac{c}{a} = 0 \\[10pt]
&\text{Complete the square:} \\[6pt]
&\left(x + \frac{b}{2a}\right)^2 = \frac{b^2}{4a^2} - \frac{c}{a} \\[10pt]
&\text{Take the square root of both sides:} \\[6pt]
&x + \frac{b}{2a} = \pm\sqrt{\frac{b^2 - 4ac}{4a^2}} \\[10pt]
&\text{Solve for } x \text{ — the quadratic formula:} \\[6pt]
&\boxed{x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}} \\[10pt]
&\text{where the discriminant } \\[6pt]
&\Delta = b^2 - 4ac \text{ determines the roots:} \\[6pt]
&\Delta > 0 \;\text{: two distinct real roots} \\
&\Delta = 0 \;\text{: one repeated real root} \\
&\Delta < 0 \;\text{: two complex conjugate roots}
\end{aligned}
$$
